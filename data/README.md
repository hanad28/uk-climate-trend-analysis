# Data

This repository includes UK historic weather-station data used for climate trend analysis and forecasting.

## Structure

The `weather_data/` folder contains:

- `ALL-WEATHER-STATIONS.csv` with station metadata
- individual station CSV files for each weather station

## Source

The dataset is based on UK Met Office historic station data.

## Notes

The notebook reads the metadata file and station-level CSVs from `data/weather_data/`.
